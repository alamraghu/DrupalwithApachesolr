# Drupal with Apache Solr Demo

This is a demonstration project showing how to integrate Apache Solr search with Drupal.

## Features
- Docker setup for Solr
- Drupal Search API configuration
- Solr configsets for Drupal

## Getting Started

1. Start Solr:
   ```
   docker-compose -f docker/docker-compose.yml up
   ```

2. Configure Drupal with the provided search API configurations.
