# 🔎 Drupal 10 + Apache Solr Search Demo

This project demonstrates how to integrate Apache Solr with Drupal 10 using Search API.

---

## 🚀 Features
- Full-text search using Solr
- Faster indexing than database search
- Relevance boosting (title > body)
- Docker-based setup (Drupal + MySQL + Solr)

---

## 🧱 Tech Stack
- Drupal 10
- Apache Solr 8
- Search API + Search API Solr
- Docker
